🛍️ Customer Shopping Behavior Analysis
📌 Overview

This project analyzes customer shopping behavior using Python, SQL (PostgreSQL), and Power BI.
It demonstrates an end-to-end data analytics workflow including data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard development to generate business insights.

📊 Dataset
Name: Customer Shopping Behavior Dataset
Records: ~3900 customers
Features: 18 columns including age, gender, purchase amount, product category, shipping type, and review ratings.
🛠️ Tools & Technologies
Python – Pandas, NumPy (Data Cleaning & EDA)
PostgreSQL – Data Storage & SQL Queries
Power BI – Interactive Dashboard
Jupyter Notebook – Analysis
Gamma & PowerPoint – Presentation & Reporting

🔄 Project Workflow
Loaded dataset using Python (Pandas)
Cleaned and prepared data
Performed EDA to identify patterns
Stored data in PostgreSQL
Wrote SQL queries to analyze customer behavior
Built an interactive Power BI dashboard
Created final report and presentation

📈 Dashboard Highlights
Total Revenue Analysis
Category-wise Sales
Customer Segmentation
Discount Usage Insights
Shipping Type Comparison
Top Purchased Products

📊 Key Insights
Subscription customers showed higher spending patterns
Discount usage increased purchase frequency
Loyal customers contributed significantly to total revenue
Certain product categories consistently received higher ratings

▶️ How to Run
Load dataset in Python using Pandas
Clean data and export to PostgreSQL
Run SQL queries for analysis
Open Power BI dashboard (.pbix)
Review report and presentation files
